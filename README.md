# Jogo da Memória: Noite de Halloween

Jogo da memória simples, com temática de Dia das Bruxas. Baseado no Desafio de Projeto da plataforma DIO.

## Tecnologias Utilizadas

- HTML
- CSS
- Javascript
- Manipulação do DOM
